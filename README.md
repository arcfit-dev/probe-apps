# Probe Landing Page

AI lead automation landing page for real estate brokers. Built with React 19, Vite, and Framer Motion.

## Tech Stack

- **React 19** — Component-based UI
- **Vite** — Fast dev server and optimized builds
- **Framer Motion** — Scroll-triggered animations, spring physics, staggered reveals
- **Lucide React** — Icon system

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```

## Project Structure

```
src/
├── components/       # 15 section components (Nav, Hero, Trust, etc.)
├── hooks/            # Animation variants and utilities
├── index.css         # Design tokens (colors, typography, spacing)
├── App.jsx           # Root layout
└── main.jsx          # Entry point
```
